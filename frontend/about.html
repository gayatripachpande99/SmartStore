About The Project

The Vision

We built SmartStore because physical stores don't have the same kind of analytics that online websites get. Usually, store owners just have to guess what's working.

Using regular CCTV cameras, we can track how customers move and where they stop, giving small businesses access to big-data insights.

We also wrote a Python script that acts like an automated assistant. It looks at the traffic and dwell times to suggest layout changes, rather than making the store owner guess what the data means.

The Technology Stack

We built SmartStore AI using a combination of cutting-edge computer vision models and reliable web technologies, carefully structured for a hackathon environment.
